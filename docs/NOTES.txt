Useful commands:
--------------------------------------------------------------------------------

sudo dnf install meson ninja-build
sudo dnf install libdnf5-devel
sudo dnf install gtk4-devel
sudo dnf install dnf5daemon-server polkit
sudo dnf install catch-devel

sudo dnf --disablerepo='rpmfusion*' install gtk4-devel

--------------------------------------------------------------------------------
Build and test:
--------------------------------------------------------------------------------

Meson handles the real build logic.
The Makefile is a thin task runner for common commands.

Native C++ tests:

export DEBUG_TRACE=y

make test

Docker app run:

make dockerrun

Docker offline test:

make dockerofflinetest

Docker Swedish translation test:

LANG=sv_SE.UTF-8 LANGUAGE=sv make dockerrun

--------------------------------------------------------------------------------
Native Polkit test:
--------------------------------------------------------------------------------

make
./dnfui

Run dnfui as a regular desktop user and apply a harmless transaction to test the
desktop Polkit prompt through dnf5daemon.

Use native Fedora for the real desktop Polkit prompt.
Use Docker for container smoke testing with a system bus and dnf5daemon.

--------------------------------------------------------------------------------
Transaction overview:
--------------------------------------------------------------------------------

The GUI runs as the regular user.
Preview and apply go through DNF5 dnf5daemon.
dnf5daemon owns the privileged package operation and Polkit behavior.
